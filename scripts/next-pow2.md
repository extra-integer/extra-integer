Get next [power-2] of [32-bit integer] *([bit-shift method])*.

```javascript
const nextPow2 = require('@extra-integer/next-pow2');
// nextPow2(integer)

nextPow2(63);
// 64
nextPow2(128);
// 128
nextPow2(1);
// 1
```

[power-2]: https://en.wikipedia.org/wiki/Power_of_two
