Get [power-10] of [32-bit integer] *(lookup method)*.

```javascript
const pow10 = require('@extra-integer/pow10');
// pow10(<integer>)

pow10(4);
// 10000
pow10(2);
// 100
pow10(-1);
// 0.1
```
